# 使用指南

## 快捷键

| 快捷键 | 功能 | 说明 |
|-------|------|------|
| `Ctrl+Shift+D` | 听写模式 | 将语音转为润色后的书面文字 |
| `Ctrl+Shift+T` | 翻译模式 | 将语音翻译为目标语言 |
| `Ctrl+Shift+A` | AI 助手模式 | 将语音作为自由指令发送给 AI |
| `Esc` | 取消录音 | 取消当前进行中的录音 |

---

## 四种模式

### 1. 听写模式（Dictate）

**用途**：将口语化的语音输入转为规范的书面文字。

**流程**：
1. 按下 `Ctrl+Shift+D` 开始录音
2. 悬浮栏出现，显示录音状态
3. 再次按下快捷键或点击停止按钮
4. AI 自动润色转录文本（去除口语填充词、修正语法）
5. 润色后的文字自动插入到当前光标位置

**适用场景**：撰写邮件、编辑文档、聊天消息

### 2. 翻译模式（Translate）

**用途**：将语音识别后直接翻译为目标语言。

**流程**：
1. 按下 `Ctrl+Shift+T` 开始录音
2. 用任意语言说话
3. 停止录音后，AI 翻译为设置中配置的目标语言
4. 翻译结果插入到光标位置

**配置**：在设置中指定目标语言（默认：English）

### 3. AI 助手模式（AI Assistant）

**用途**：将语音作为自由形式的 AI 指令。

**流程**：
1. 按下 `Ctrl+Shift+A` 开始录音
2. 说出你的请求（如"帮我写一封请假邮件"）
3. 停止录音后，AI 根据指令生成内容
4. 结果显示在结果面板中，可复制或插入

**特性**：
- 结果面板支持 Markdown 渲染
- 可点击「复制」或「插入到光标」

### 4. 编辑模式（Edit）

**用途**：用语音指令修改已选中的文本。

**流程**：
1. 在任意应用中选中一段文本
2. 按下快捷键并说出修改指令（如"把这段改成更正式的语气"）
3. AI 根据指令修改选中的文本

---

## 悬浮栏

悬浮栏是录音和处理过程中的主要交互界面：

| 状态 | 显示内容 |
|------|---------|
| 录音中 | 当前模式 + 音量条 + 停止/取消按钮 |
| 处理中 | 加载动画 |
| 完成 | 成功提示（自动消失） |
| 错误 | 错误信息 + 重试按钮 |

---

## 历史记录

所有转录和处理结果都会自动保存到本地数据库。在设置窗口中：

- 查看最近的转录历史
- 搜索历史记录
- 删除单条或批量删除

---

## 离线队列

当网络不可用时：

1. 录音会正常保存到本地
2. 录音自动加入离线队列（SQLite 持久化，重启不丢失）
3. 网络恢复后自动处理队列中的录音
4. 应用每 30 秒检测一次网络状态
