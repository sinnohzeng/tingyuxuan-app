# 使用指南

## 快捷键

| 快捷键 | 功能 | 说明 |
|-------|------|------|
| `RAlt` | 听写模式 | 将语音转为润色后的书面文字 |
| `Shift+RAlt` | 翻译模式 | 将语音翻译为目标语言 |
| `Alt+Space` | AI 助手模式 | 将语音作为自由指令发送给 AI |
| `Esc` | 取消录音 | 取消当前进行中的录音 |

**平台兼容性注意事项**：
- `Alt+Space` 在 Windows 上是系统快捷键（窗口菜单），如被系统拦截可在设置 → 快捷键中自定义
- Linux 下 RAlt 可能被配置为 Compose 键，需在系统输入设置中调整
- `Modifiers::SHIFT` 不区分左右 Shift，因此左 Shift + RAlt 也会触发翻译模式

## 四种模式

> 当前 MVP 限制：单次录音时长 **<= 5 分钟**（仅使用 `qwen3-asr-flash`）。

### 1. 听写模式（Dictate）

**用途**：将口语化的语音输入转为规范的书面文字。

**流程**：
1. 按下 `RAlt` 开始录音
2. 悬浮栏先进入 `starting` 微加载（几十毫秒）
3. 录音流就绪后进入 `recording`，显示音量波形
4. 到第 4 分钟会出现 60 秒倒计时与时长限制提示
5. 再次按下快捷键或点击停止按钮（若未手动停止，5 分钟会自动停止）
6. 进入 `thinking`（思考中），多模态 AI 一步完成语音识别和润色
7. 处理后的文字自动插入到当前光标位置

**适用场景**：撰写邮件、编辑文档、聊天消息

### 2. 翻译模式（Translate）

**用途**：将语音直接翻译为目标语言。

**流程**：
1. 按下 `Shift+RAlt` 开始录音
2. 用任意语言说话
3. 停止录音后，多模态 AI 一步识别并翻译为设置中配置的目标语言
4. 翻译结果插入到光标位置

**配置**：在设置中指定目标语言（默认：English）

### 3. AI 助手模式（AI Assistant）

**用途**：将语音作为自由形式的 AI 指令。

**流程**：
1. 按下 `Alt+Space` 开始录音
2. 说出你的请求（如"帮我写一封请假邮件"）
3. 停止录音后，AI 根据指令生成内容
4. 结果显示在结果面板中，可复制或插入

**特性**：
- 结果面板支持 Markdown 渲染
- 可点击「复制」或「插入到光标」

### 4. 编辑模式（Edit）

**用途**：用语音指令修改已选中的文本。

**流程**：
1. 在任意应用中选中一段文本
2. 按下快捷键并说出修改指令（如"把这段改成更正式的语气"）
3. AI 根据指令修改选中的文本

## 悬浮栏

悬浮栏是录音和处理过程中的主要交互界面：

| 状态 | 显示内容 |
|------|---------|
| 启动中 (`starting`) | 微加载动画（录音引擎初始化） |
| 录音中 | 当前模式 + 上下对称波形 + 停止/取消按钮 |
| 录音倒计时（第 4 分钟后） | 60 秒倒计时 + 时长限制 Pop up |
| 思考中 (`thinking`) | 加载动画（AI 正在处理） |
| 完成 | 成功提示（自动消失） |
| 错误 | 错误信息 + 操作建议 |

## 历史记录

所有转录和处理结果都会自动保存到本地数据库。在设置窗口中：

- 查看最近的转录历史
- 搜索历史记录
- 删除单条或批量删除

---

## 离线队列

当网络不可用时：

1. 录音会正常保存到本地
2. 录音自动加入离线队列（SQLite 持久化，重启不丢失）
3. 网络恢复后自动处理队列中的录音
4. 应用每 30 秒检测一次网络状态
