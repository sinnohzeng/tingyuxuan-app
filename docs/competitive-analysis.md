# 竞品分析

> 版本：v1.0 | 更新日期：2026-02-28

## 1. Typeless 产品分析

[Typeless](https://typeless.so/) 是目前最具代表性的 AI 语音输入工具，主打"用说话代替打字"。

### 定价模型

| 方案 | 价格 | 额度 |
|------|------|------|
| Free | $0/月 | 2000–4000 词/周 |
| Pro（年付） | $12/月 | 无限 |
| Pro（月付） | $30/月 | 无限 |

### 与听语轩的核心差异

Typeless 采用 SaaS 订阅制，用户依赖其账号体系和云端服务。听语轩采用 **BYOK（Bring Your Own Key）** 模式——用户自带 API Key，不绑定任何 SaaS 账号，使用成本完全透明可控。

## 4. 听语轩差异化定位

- **用户拥有 API Key**：BYOK 模式，无订阅费用，按实际用量付费，成本透明
- **隐私优先**：零数据留存，音频和文本不经过第三方中转服务器
- **全平台覆盖**：Windows、macOS、Linux、Android——业界唯一支持 Linux 的同类产品
- **开源架构**：代码公开，可审计、可自部署，社区驱动迭代
- **多模态单步管线**：音频直送多模态 LLM，省去 STT 中间环节，架构更简、延迟更低
